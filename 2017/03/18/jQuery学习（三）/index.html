jQuery事件

页面对不同访问者的响应叫做事件
事件处理程序指的是当 HTML 中发生某些事件时所调用的方法

常见的jQuery事件

  • $(document).ready() 方法允许我们在文档完全加载完后执行函数
  • click() 鼠标点击事件
  • dbclick() 鼠标双击事件
  • mouseenter() 鼠标指针划过该元素时触发
  • mouseleave() 鼠标离开元素触发
  • mousedown() 鼠标移动到元素上方，并按键按下时触发
  • mouseup() 鼠标移动到元素上方，并按键按下时离开触发
  • hover() 模拟光标悬停事件
  • focus() 当元素获得焦点时，发生 focus 事件。
  • blur() 元素失去焦点触发
